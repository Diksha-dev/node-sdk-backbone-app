Demo App for Node.js SDK

This application demonstrates the Salesforce Marketing Cloud's Fuel Node.js SDK, providing access to a variety of Salesforce Marketing Cloud objects.

ATTENTION: The default mode of this app returns code examples to the UI if no credentials are supplied. If you run this app with credentials, do not deploy it to a public URL unless the app is password protected.

Objects

Choose a Salesforce Marketing Cloud object.

Methods

Choose the function to perform on the object.

NOTE: Some tests may need to be modified (in the code) to work with your particular MC account data.

API Connections

You can access the SOAP and REST node libraries directly from the SDK.

NOTE: These may be needed if the object you need is not included in the SDK.
